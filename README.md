# 🚀 Employee Management System (MERN Stack)

A full-stack Employee Management System built using MERN stack with authentication, CRUD operations, and cloud deployment.

---

## 🌐 Live Demo
Frontend: https://employee-management-system-gray-gamma.vercel.app/  
Backend: https://employee-management-system-2-d3rq.onrender.com  

---

## ⚙️ Tech Stack

- Frontend: React (Vite), Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JWT
- Deployment: Vercel + Render

---

## ✨ Features

- User Login & Authentication (JWT)
- Add / Edit / Delete Employees
- Search by Name / Department
- Dashboard with:
  - Total Employees
  - Total Salary
  - Departments Count
- Protected Routes (Login required)
- Responsive UI

---

## 📦 API Endpoints

### Auth
- POST /api/auth/login

### Employees
- GET /api/employees
- POST /api/employees/add
- PUT /api/employees/:id
- DELETE /api/employees/:id

---

## 🚀 Deployment

- Frontend deployed on Vercel
- Backend deployed on Render
- Database hosted on MongoDB Atlas
